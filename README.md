# [https://nor.the-rn.info](https://nor.the-rn.info)

## Updating the Discography

```zsh
npm run discography
```

## Install

```zsh
git clone git@github.com:tyleretters/nor.the-rn.info.git
gem install bundler -v 2.5.22
bundle install
npm install -g sass
npm i
```

## Development

```zsh
npm run dev
npm run css
```

## Build

```zsh
npm run build
```
